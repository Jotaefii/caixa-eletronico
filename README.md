# 💳 Caixa Eletrônico (ATM Simulator)

Projeto simples de simulação de um **caixa eletrônico**, desenvolvido em **linguagem C** com o objetivo de praticar **lógica de programação** e **estrutura de menus interativos no terminal**.

## 📁 Estrutura do projeto

```bash
atm-simulator
│
├── sistema
│   └── sistema.c
│
├── README.md
└── .gitignore
```

## 📚 Objetivo

Este projeto foi criado para treinar conceitos fundamentais de programação, como:

- Estruturas condicionais
- Estruturas de repetição
- Organização de funções
- Manipulação de string

## ⚙️ Funcionalidades

O sistema possui um menu interativo com as seguintes operações:

- 📋 Menu principal
- 💰 Consultar saldo
- ➕ Realizar depósito
- ➖ Realizar saque

## 🛠 Tecnologias utilizadas

- Linguagem **C**
- Compilador **GCC**

## ▶️ Como executar o projeto

1. Clone o repositório

```bash
git clone https://github.com/Jotaefii/atm-simulator.git
```

2. Entre na pasta do projeto

```bash
cd atm-simulator
```

3. Compile o programa

```bash
gcc sistema/sistema.c -o atm
```

4. Execute o programa

```bash
./atm
```
